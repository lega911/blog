Свежий nginx на ubuntu

2011-01-17 nginx ubuntu web server linux install

Пример установки для ubuntu maverick (10.10)

Добавляем 2 строки в файл /etc/apt/sources.list

deb http://ppa.launchpad.net/nginx/stable/ubuntu maverick main
deb-src http://ppa.launchpad.net/nginx/stable/ubuntu maverick main

Добавляем ключ
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys C300EE8C

Устанавливаем
sudo apt-get update
sudo apt-get install nginx-full
