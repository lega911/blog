linux, аргумент предыдущей команды

2010-07-31 linux bash

Пример, выполняем команду из 2-х аргументов tail и /etc/group, следующей команде что-б поставить аргумент нужно указать !:N - где N номер аргумента начиная с 0.

tail /etc/group
...
nano !:1
nano /etc/group

Последний аргумент предыдущей команды
nano !$
а если нажать Alt + '.' (Альт + точка на символьной клавиатуре) то предыдущий аргумент сразу подставиться
