Базовая настройка Grunt для сборки и минификации JavaScript, CoffeeScript и CSS

2013-12-15 gruntjs javascript coffeescript uglify

Пример базовой настройки Grunt для сборки и минификации JavaScript, CoffeeScript и CSS.

Устанавливаем grunt (из под root)

# npm install grunt-cli -g

Настраиваем приложение (создается package.json)
$ npm init

Устанавливаем необходимые пакеты
$ npm install grunt --save-dev
$ npm install grunt-contrib-coffee grunt-contrib-concat grunt-contrib-uglify grunt-contrib-cssmin grunt-contrib-connect --save-dev


Создаем Gruntfile.js файл и настраиваем, см. пример

Сборка проекта
$ grunt

Запуск http тест-сервера
$ grunt connect:main:keepalive
